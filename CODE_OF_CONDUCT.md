# Code of Conduct

## Our pledge

We want participation in this project to be a respectful, productive experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Focus on technical merit and reproducibility.
- Keep discussions on-topic and specific.
- Assume good intent, ask clarifying questions, and offer actionable suggestions.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insults, or intentionally disruptive behavior.
- Sharing private information without permission.

## Scope

This code of conduct applies to interactions in:

- GitHub issues and pull requests
- code review discussions
- project documentation and related communication channels

## Enforcement

The maintainer is responsible for clarifying expectations and taking appropriate action, which may include:

- requesting changes to tone/content
- locking conversations
- closing issues/PRs
- blocking users in severe cases

## Contact

If you experience or witness unacceptable behavior, contact the maintainer privately (see the repository “About” section or maintainer profile).
